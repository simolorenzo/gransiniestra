Starter limpio

Un repo simple para guías y mapa en GitHub Pages

Sin CMS, sin login raro y con una estructura pensada para que la parte más delicada del contenido esté controlada. Las guías se editan en Markdown y el mapa usa un CSV simple.

Ver guías Abrir mapa

Para guías

Cada guía es un archivo Markdown. Fácil de duplicar, revisar y versionar.

Para el mapa

Los puntos salen de un CSV sencillo y hay una utilidad para sacar coordenadas haciendo clic.

Índice claro

La portada lleva a las guías, el mapa y la utilidad de edición del mapa.

Mantenimiento simple

Todo vive en un solo repo y se publica en GitHub Pages sin dependencias raras.

1. Editar una guía

Abre un archivo en content/guias/ y cambia el texto.

2. Añadirla al índice

Añade una línea nueva en data/guias.csv.

3. Colocarla en el mapa

Añade o edita su fila en data/mapa.csv usando la utilidad de coordenadas.

Hecho para GitHub Pages. Sin backend. Sin Netlify. Sin Decap.
